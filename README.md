# bmw-clone - landing page website clone

This project is an attempt to create a landing page stylised as to be the clone of product page of certain car manufacturer.

Note: This project was done for educational purposes and to showcase certain competencies. I do not claim any rights to trademarks and products which may have been presented.

## Live demo

Live demo available: https://car-site.netlify.app/

## Technologies
* HTML5
* CSS/Sass (BEM)
* Javascript ES6
* React

## Installation

To install, you need to clone the repository and install all used packages using the command:

```
npm install
```

Then run the project with:

```
npm start
```

To build, run:

```
npm run build
```
